Kirby Super Star Practice Hack

A ROM hack for KSS to streamline speedrun practice. This hack is intended for JP v1.0.

You will need LunarIPS and a ROM file to apply the patch.

This hack is currently in "beta", and will be officially complete once I finish the list of features I would like to have done in "Features Checklist" (or when I am satisfied or tired of working on it).

Majority of this hack has been done by Nippo. Contact me if you would like to report a bug.

If you are using an FxPak and are experiencing large lag spikes, put these files in your sd2snes folder (if your flashcart revision is old enough, you can alternatively use this firmware).

Features list:

  • Infinite lives
  • Death animation removed
  • Mute music by using Stereo/Mono button
  • Defeated Boss screen flashing removed (a toggle for this will be added later)
  • Quick Save Quick Load:
    • R + Select: Create Save State
    • L + Select: Load Save State
  • Quick room reset with R + X
    • Data such as ability and RNG is saved on room reset. To not save these values upon room reset, do L + R + X.
  • Powers, Helper, and Wheelie Rider state are kept through death.
  • Instant 100% save file with Up + X on file select screen (selected file will be made 100%)
  • Quick select abilities using the following buttons:
    • Holding R + A:
      • Up: Jet
      • Right: Wheel
      • Down: Hammer
      • Left: Plasma
      • Select: Remove Ability
    • Holding L + A:
      • Up: Wing
      • Right: Stone
      • Down: Suplex
      • Left: Cutter
      • Select: Remove Helper
    • Holding L + R + A
      • Left: Cycle left through all abilities
      • Right: Cycle right through all abilities
  • Enable Free Movement with L + R, then Y
    • This code does not work in many situations, such as when riding on Wheelie and during cutscenes. Bugs still need to be fixed.
  • Milky Way Wishes ability presets:
    • Toggling this feature makes it so upon entering a level, you are automatically given whatever abilities you would normally have upon entering that level during a speedrun. Pressing Select on the MWW map screen allows you to cycle through these options:
      • 0: Off
      • 1: Any%
      • 2: Any% Plasma
      • 3: 100%
    • Note: There is no visual indication for when this is being toggled, this still needs to be added.
  • Spring Breeze bosses always at New File HP
  • GCO bosses always respawn
  • GCO Treasures always respawn
  • MWW Abilities always respawn
  • RoMK timer always frozen
  • RoMK chapter select on subgame screen
  • RoMK dialogue cutscenes skippable by pressing Start
  • Use L and R to quick select planets in MWW, hold Start to double movement speed
  • Fatal game crashes will attempt to automatically reset the game to avoid data loss
  • Custom HUD Version Features (Note that in this version, most features above do not exist):
    • View specific HP values
    • Input display
    • Frame counter
    • RNG values
    • Jet / Plasma charge
  • Change Kirby's colors by holding Select + X, then pressing L or R.
